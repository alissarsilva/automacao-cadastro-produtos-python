# Automação de Cadastro de Produtos com Python

Este projeto realiza a automação do cadastro de produtos em um sistema web utilizando Python.

A automação lê uma base de dados em CSV e cadastra automaticamente cada produto no sistema, simulando ações do usuário no teclado e mouse.

## Tecnologias utilizadas

- Python
- PyAutoGUI
- Pandas

## Como funciona

O script executa as seguintes etapas:

1. Abre o navegador automaticamente
2. Acessa o sistema de cadastro
3. Realiza login no sistema
4. Lê uma base de dados em CSV
5. Percorre cada linha da tabela
6. Cadastra os produtos automaticamente no sistema

## Estrutura do projeto

main.py → script de automação
produtos.csv → base de dados com os produtos
requirements.txt → bibliotecas utilizadas

## Objetivo do projeto

Projeto desenvolvido para praticar automação de tarefas com Python, utilizando leitura de dados e automação de interface.

## Observação

As posições de clique utilizadas no script foram definidas de acordo com a resolução da tela utilizada durante o desenvolvimento.
